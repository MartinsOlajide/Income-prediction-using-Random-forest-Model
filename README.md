# Income Prediction & Analysis

## 📊 Project Overview
This project analyzes census demographic data to understand factors influencing whether an individual earns more than $50K per year. It combines exploratory data analysis and machine learning to uncover patterns and build a predictive model.


## 🛠️ Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  


## 🔍 Key Insights

- **Education Impact**: Individuals with higher education levels (Bachelors, Masters, Doctorate) are significantly more likely to earn above $50K. Income increases consistently with education level.

- Occupation Trends: Professional and technical roles have the highest proportion of high earners, while service and manual labor roles tend to have lower income levels.

- Gender Disparity: Males are more likely to earn above $50K compared to females, suggesting possible structural or societal income inequality.

- Work Sector Distribution: The majority of individuals work in the private sector. Self-employment is also notable, while government employment is less common but still significant.

- Relationship Status: Most individuals are married or in relationships, with fewer individuals classified as not-in-family or other relatives.


## ⚙️ Approach
- Data cleaning and preprocessing (handling missing values, encoding categorical features)  
- Exploratory Data Analysis (EDA) using visualizations  
- Feature engineering  
- Model building using Random Forest  
- Model evaluation using Accuracy, Precision, Recall, F1-score, and ROC-AUC  


## 📈 Model Performance
- Accuracy: ~82%  
- ROC-AUC Score: ~0.86  


## 📁 Dataset
The dataset used is based on census demographic data for income classification.


## 🚀 Key Learnings
- Importance of feature engineering in improving model performance  
- Real-world income disparities across education, gender, and occupation  
- Practical experience with classification models and evaluation metrics
